# Phase 1 — Project Skeleton

## Goal

Get Tauri + React + TypeScript + Vite + shadcn running with the design token foundation in place. No UI screens. No business logic. Just a working scaffold with the right dependencies and file structure.

## Success Criteria

- `npm run tauri dev` launches a desktop window
- The window shows the Phase 1 placeholder (gold "Tome of Creation" heading)
- All design tokens are defined in `tokens.css` and `tokens.ts`
- shadcn components are installed and use the warm dark palette
- Folder structure matches the spec
- No raw hex values anywhere in source

## Steps (in order)

See `Documentation/SESSION_phase_1_skeleton.md` for the exact Claude Code prompt.

## What Is NOT in Phase 1

- No navigation rail
- No actual screens
- No database
- No Tauri commands (other than SQL plugin registration)
- No AI integration
- No `.tome` file handling
